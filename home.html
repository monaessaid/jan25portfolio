Selected Work

PROJECTS

ClearGivers

ClearGivers, the gift registry for charities, connects donors with nonprofits and businesses in their communities. Redesigned the entire ClearGiver site and made backend changes to improve its efficiency. Implemented a new style guide and created custom graphics to more effectively explain the charity's goals and purpose.

Twitter API

Implemented a RESTful API for Twitter using Spring Boot, JPA, and Postgresql in a team of 3. Wrote Spring services and controllers that handle requests, perform validation and business logic, and transform data between the API and database models.

Quizler CLI

Worked on the front-end of a Spotify quiz site where players can guess which artists are popular on Spotify. Players enter their name, then select the quiz's difficulty level and number of questions. Their scores are displayed on the leaderboard.

Quizler CLI

Created a custom library with Javascript, inquirer.js, and vorpal.js for a command line application. It builds quizzes and allows the user to take quizzes they have built or pull random questions from multiple quizzes to take a random quiz.

Three.js Galaxy

Model of a galaxy created with Three.js. Each of the large planets has its own orbit and image-based textures to create a realistic effect. The smaller planets are in randomly-generated positions that change each time the page is refreshed.

Workspace WebApp

Built a web application that helps remote workers find great places to work anywhere in the world. It uses a React client and a server built with Node.js that communicates with 3 Google APIs (Autocomplete, Maps, and Places).
